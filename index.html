Welcome to the ICS Textbook Application Project

The project is intended to create a web application available for the Computer Science students at the University of Hawaii at Manoa to sell and buy used computer science textbooks. These textbooks are books only used in the classes offered at the Information and Computer Science department at Manoa.

NOTE

This project is under work so it is not available yet for deployment.

Overview

  • Students can either: "Make an offer" to sell books, or "Make a request" to buy books.
  • Books are pre-defined, making it more neatly the web application. For example, it makes easier to search for books.

Functional Elements

As of 5/8/2013, "Making an offer" and "Making a request", both work properly. Both were tested also using FluentLinium. Also, editing and deleting either an offer or request works.

To be implemented

  • Ideally, we want a registration form for students to use the web application. We need also to identify that the students are from the Computer Science department at UHM.
  • Search for offers and requests is not yet implemented. It should be a top priority since it is an essential feature for the web application.
